Linux 是什么？

Linux 是一种自由和开放源码的类 Unix 操作系统。

它开放源代码，任何人都可以自由地使用源代码，甚至修改和再发布，前提是遵循 GPL 。

它是在 Minix（一个迷你版本的类 Unix 操作系统） 的启发下创作的，它是类 Unix 操作系统，但它不是 Unix 操作系统。

它是一个操作系统，类似于 Windows，能够完成 Windows 的大部分工作。

Linux 能做什么？

它能够完成 Windows 的大部分工作，它能够做到的事情很多，下面只列举 Linux 的一部分应用场景。

服务器

Linux 在服务器操作系统的占有率达到了 60% 以上，超过了 Windows。

甚至 Linux 成为了微软 Azure 上最流行的操作系统，超过一半的 Azure VM 是 Linux。

我们平时浏览网页，其中有很多网页的后台采用了 LNMP 架构，服务器运行在 Linux 上。

嵌入式设备

Linux的低成本、强大的定制功能以及良好的移植性能，使得Linux在嵌入式系统方面也得到广泛应用。

我们平时使用路由器拨号上网，其中有很多路由器是基于 OpenWrt（专为嵌入式设备定制的Linux系统）制作的，比如说小米路由器。

智能移动设备

在智能手机、平板电脑等移动设备方面，Linux也得到重要发展，基于 Linux 内核的 Android 操作系统已经成为了全球最流行的智能手机操作系统。

甚至出现了由 Linux 操作系统驱动的智能手机，比如说 PinePhone。

桌面系统

有很多软件都有 Linux 版本，比如说 Google Chrome。

还有很多软件没有 Linux 版本，但是我们可以在 Linux 上找到大部分软件的替代品。

还有一部分软件只能在 Windows 上运行，但是我们可以通过 Wine 正常运行和工作。

甚至 Value 推出了 Proton（Wine 的一个分支），可以在 Linux 上允许 Windows 游戏，比如说 GTA5。

Linux 有什么优势？

免费

绝大多数 Linux 发行版是免费的，任何人都可以免费从官网下载、安装和使用它们。

但 Linux 并不意味着免费，少数 Linux 发行版是付费的，比如说 RHEL 。还有一部分 Linux 发行版提供付费的技术支持，比如说 Ubuntu。

开源

所有的 Linux 发行版都必须遵循 GPL ，向所有人开放源代码。

任何人都可以自由地使用源代码，甚至修改和再发布。

如果存在什么安全漏洞，都能够在第一时间内发现，并进行修复。

社区

大多数 Linux 发行版都有官方的 Wiki 和 论坛，比如说 Ubuntu Wiki 和 Ubuntu 论坛。

如果你遇到什么问题，可以在 Wiki 上 查找解决方案。或者在论坛上向其他用户提问，问题一般都能得到解决。
