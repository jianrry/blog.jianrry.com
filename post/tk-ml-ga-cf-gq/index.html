注册域名

Step1：登陆你的 Freenom 账号

打开官网，登录你的 Freenom 账号。

如果你没有 Freenom 账号，可以暂时跳过这一步骤，进入下一步。

Step2：搜索你想注册的域名

打开官网，搜索你想注册的域名。

输入搜索的关键字，然后点击“Check Availability”，检查可用性。

Step3：将域名加入购物车

点击“Get it now!”按钮，将域名加入购物车。

Step4：确认订单

点击“Checkout”按钮，进入订单页面。

默认是 3 Months @ FREE （免费3个月），你可以更改为 12 Months @ FREE（免费12个月）。

Step5：完成订单

勾选上“ I have read and agree to the Terms & Conditions "，然后点击" Complete Order " ，完成订单。

如果你没有 Freenom 账号，则输入你的邮箱，点击邮件中的链接，进入下一步。

如果提示用户已存在，你可以换一个邮箱试试，多试几次总会成功。

Step6：注册成功

如果出现了下面的提示，则说明已经注册成功。

域名解析

Step1：访问域名管理页面

点击此处，访问域名管理页面。

点击 "Manage Domain" 按钮，管理你的域名。

Step2：修改 Name Server

点击 "Management Tools" 按钮，继续点击 "Nameservers" 按钮，进入 Name Server 管理页面。

选中 "Use custom nameservers(enter below)"，自定义 Name Server。

输入 Name Server 的地址，然后点击 "Change Nameservers"。

如图所示，将 jianrry.cf 托管到 cloudflare 。

Step3：添加 DNS 记录

你可以参考这篇文章，添加 DNS 记录。

域名续期

在你的域名到期前14天，你可以访问域名续期页面，手动续期。

也可以基于这个项目，自动续期。

注意事项

  1. 国内的 IP 已经无法注册 Freenom 的域名，请使用其他区域的 ip 进行注册。
  2. 你的 ip 地址必须和 Freenom 个人资料的地址保持一致，才能注册成功。比如说 你使用香港 ip，Freenom 个人资料的地址可以填写 香港迪斯尼乐园 。
  3. Freenom 的免费域名仅供临时使用，随时有可能被收回。如果你的域名访问量稳定，且达到了一定阈值 或者 有人想要购买你的域名，Freenom 则会收回你的域名。
