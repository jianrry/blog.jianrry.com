下载 IntelliJ IDEA

IntelliJ IDEA 官网：https://www.jetbrains.com/zh-cn/idea/

IntelliJ IDEA Ultimate 2021.2：https://download.jetbrains.com/idea/ideaIU-2021.2.win.zip

IntelliJ IDEA 分为2个版本：Ultimate（企业版）和 Community（社区版）。企业版需要付费才能使用，可以免费试用30天，可以使用全部的功能。社区版无需付费就能使用，可以永久免费试用，只能使用部分的功能。

根据你的操作系统，下载对应平台的最新版本的 IntelliJ IDEA，建议使用企业版。

比如说，我的操作系统是 Windows x64，下载的是 ideaIU-2021.2.win.zip。

安装 IntelliJ IDEA

将已下载的安装包，解压到指定的安装目录，比如说 D:\App\IntelliJ IDEA。

进入安装目录的 bin 目录，为 idea64.exe 创建桌面快捷方式。

双击桌面快捷方式，就可以启动 IntelliJ IDEA 了。

下载 & 安装插件

打开 Plugins，在 Marketplace 中输入关键字，比如说 Chinese。

选中 Chinese ​(Simplified)​ Language Pack / 中文语言包，然后点击 Install 按钮，会自动下载并安装该插件。

点击 Restart IDE，重启 IntelliJ IDEA，会发现界面已经显示中文。
