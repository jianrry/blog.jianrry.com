Java 入门指南（二）：Hello World

2019-03-09

本文是《Java 入门指南》的第二篇文章，介绍了 Java 的第一个程序：Hello World。

    • Step1：新建 Java 源文件
    • Step2：编译 Java 源文件
    • Step3：运行 Java 字节码
    • 总结

Step1：新建 Java 源文件

打开文本编辑器，输入下面的代码，另存为 Hello.java，保存在我的文档 。

public class Hello {

    public static void main(String[] args){

        System.out.println("Hello World!");

    }

}

Step2：编译 Java 源文件

进入我的文档，然后打开命令提示符。输入下面的代码，然后回车。

javac Hello.java

如果 Java 源文件没有任何错误，则不会有任何输出，则当前目录下会生成一个名为 Hello.class 的字节码文件。

Step3：运行 Java 字节码

进入我的文档，然后打开命令提示符。输入下面的代码，然后回车。

java Hello

如果出现下面的提示，则说明 Java 源文件没有任何错误。

Hello World!

总结

  1. 一个 Java 源文件只能有一个公共类，并且公共类名必须与文件名完全一致。
  2. javac 可以将 Java 源文件编译为 Java 字节码。
  3. java 可以运行 Java 字节码。
