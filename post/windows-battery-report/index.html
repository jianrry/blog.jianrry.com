Windows 查看电池使用时间报告

2019-01-28

本文以 Windows 10 为例，介绍了如何查看电池使用时间报告，判断电池的健康状况。

    • Step1：打开命令提示符
    • Step2：生成电池使用时间报告
    • Step3：查看电池使用时间报告

Step1：打开命令提示符

按下 "Windows + R" 键，打开 "运行" 。然后输入 "cmd" ，并按下回车键。

Step2：生成电池使用时间报告

打开命令提示符，输入下面的代码，并按下回车键。

powercfg /batteryreport

将会生成一份电池使用时间报告，并保存在 C:\Users\用户名\battery-report.html 目录下。

Step3：查看电池使用时间报告

进入报告所在的目录下，选择用浏览器打开 battery-report.html，就可以查看电池使用时间报告了。

如果 FULL CHARGE CAPACITY（满电容量）远小于 DESIGN CAPACITY（设计容量），则说明电池衰减可能很严重了，可能需要更换电池了。
