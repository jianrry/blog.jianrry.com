Windows 快捷键汇总

2019-01-26

Windows 键盘快捷键就是按键或按键组合，可以替代鼠标执行的操作，比如说录屏或截图等，提高工作效率。

    • Ctrl 键
    • Windows 键
    • 其他键
    • 参考资料

Ctrl 键

快捷键 描述
Ctrl + A 全选
Ctrl + C 复制
Ctrl + V 黏贴
Ctrl + X 剪切
Ctrl + Z 撤销
Ctrl + Y 恢复（恢复到上一步操作，与撤销相反）
Ctrl + F 搜索
Ctrl + H 替换
Ctrl + S 保存
Ctrl + D 移动到回收站
Ctrl + N 打开新窗口
Ctrl + R 刷新活动窗口
Ctrl + W 关闭活动窗口
Ctrl + Alt + Tab 切换活动的应用
Ctrl + Shift + Esc 打开任务管理器

Windows 键

快捷键 描述
Windows 打开或关闭开始菜单
Windows + A 打开“快速操作”面板
Windows + B 光标移动到系统托盘（然后按下“Enter”键，可以显示系统托盘里面的图标）
Windows + D 显示或隐藏桌面
Windows + E 打开资源管理器
Windows + F 提交反馈
Windows + G 打开或关闭 Xbox Game Bar
Windows + H 语音听写
Windows + I 打开设置
Windows + K 投屏到其他设备
Windows + L 锁屏
Windows + M 窗口最小化
Windows + P 切换投影模式
Windows + Q/S 打开搜索
Windows + R 打开“运行”
Windows + V 打开剪贴板
Windows + X 打开菜单
Windows + 空格 切换输入法
Windows + Tab 切换虚拟桌面
Windows + Ctrl + D 新建虚拟桌面
Windows + PrtSc 截屏
Windows + Shift + PrtSc 专业截屏

其他键

快捷键 描述
Num Lock 打开或关闭小键盘锁
CapsLk 切换英文大小写
