Linux 发行版是什么？

Linux 发行版是指由一些组织、团体、公司或者个人制作并发行的 Linux 版本。

包括了一个Linux 内核，来自GNU的工具和库，和附加的软件、文档，还有一个窗口系统，窗口管理器，和一个桌面环境等。

目前有超过 300 个 Linux 发行版，其中最流行的 Linux 发行版有十几个，较为知名的有 Debian、Ubuntu、Fedora、CentOS、Arch Linux 和 openSUSE 等。

一些流行的 Linux 发行版

按打包方式划分，下面列举了一些流行的 Linux 发行版。

Debian 系

Debian 及其派生发行版使用 deb 软件包格式，并使用 dpkg 及其前端作为软件包管理器。

  • Debian ：以稳定性闻名，很多服务器都使用 Debian 作为其操作系统，是最流行的 Linux 发行版之一。
  • Ubuntu：基于 Debian，使用 GNOME 桌面环境，是最流行的 Linux 发行版之一。
    • Ubuntu GNOME：基于 Ubuntu，使用 GNOME 桌面环境的 Ubuntu。
    • Kubuntu：基于 Ubuntu，使用 KDE 桌面环境的 Ubuntu。
    • Lubuntu：基于 Ubuntu，使用 LXDE 桌面环境的 Ubuntu。
    • Xubuntu：基于 Ubuntu，使用 Xfce 桌面环境的 Ubuntu。
    • Linux Mint：基于Ubuntu，旨在提供开箱即用的用户体验，预装了浏览器、多媒体编解码器、Java 等软件。
    • Ubuntu Kylin：基于 Ubuntu，由 Canonical 公司与中国工信部联合开发，添加了少量中国化定制的 Ubuntu 。
  • Deepin：基于Debian，使用自行开发的Deepin DE桌面环境，预装了大量国产软件，更符合国人的使用习惯。
    • 统一操作系统（UOS）：基于Deepin，由中国电子集团、武汉深之度科技有限公司、南京诚迈科技、中兴新支点在内的多家中国公司联合开发的商业化发行版。

Red Hat 系

Red Hat Linux 和 openSUSE 是最早使用 RPM 格式软件包的发行版，如今 RPM 格式已广泛运用于众多的发行版。

  • Red Hat Enterprise Linux：由 Red Hat 公司开发，是 Red Hat Linux 的企业版，是最流行的 Linux 发行版之一。
  • Fedorax：由 Red Hat 公司和其社区共同开发，是 Red Hat Enterprise Linux 发行版的上游，被认为可用的技术最终会加入到 Red Hat Enterprise Linux 中。
  • CentOS：由 CentOS 社区开发，旨在与 Red Hat Enterprise Linux 100% 兼容，但不包含 Red Hat 的商业软件。
  • openSUSE：虽然采用了 Red Hat 系的RPM软件包，但并未从 Red Hat 继承代码或架构。

Arch 系

Arch Linux 使用 .pkg.tar.xz 格式的软件包，并使用 Pacman 作为软件包管理器。

  • ArchLinux：针对x86-64的CPU做了优化，采用滚动更新模式，适合动手能力较强的用户。
  • Manjaro：基于 Arch Linux，使用 Xfce 、GNOME和 KDE Plasma 作为桌面环境，让强大的 Arch Linux 更方便用户使用。
  • SteamOS：基于 Arch Linux，使用 KDE Plasma 作为桌面环境。由 Value 公司开发，是 Steam Machine 与 Steam Deck 所使用的操作系统。

选择合适的Linux发行版

Zegenie Studios 提供了 Linux 发行版选择器，通过完成一个不超过十六题的问卷测试，你就可以选择合适的 Linux 发行版了。

如果你通过上面的问卷测试，仍然没有合适的 Linux 发行版，以下的 Linux 发行版或许适合你。

如果你刚接触 Linux ，我建议你选择 Debian 系的 Linux 发行版，比如说 Ubuntu 等。

如果你熟练使用 Linux，你可以尝试 Arch 系的 Linux 发行版，比如说 Manjaro 等。

如果日常使用国产软件，不想花费时间和精力折腾，我建议你选择 Deepin。
