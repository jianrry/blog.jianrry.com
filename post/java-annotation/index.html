单行注释

描述：

只能注释一行。

用法：

//注释内容

实例：

//这是一行注释。

多行注释

描述：

可以注释多行。

用法：

/*
注释内容
*/

实例：

/*
这是第一行注释。
这是第二行注释。
这是第三行注释。
*/

文档注释

描述：

在程序中嵌入关于程序的信息，更加方便地记录程序信息。

可以使用 javadoc 生成信息，并输出到 html 中。

用法：

/**
*注释内容
*/

实例：

/**
 * @ClassName Test
 * @Description TODO
 * @Author Jianrry
 * @CreateTime 2021/12/8 11:26
 * @Version 1.0.0
 */
