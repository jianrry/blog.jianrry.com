NameSilo 修改 NS 服务器

2019-02-05

NameSilo 域名的 DNS 记录生效很慢，建议修改域名的 NS 服务器，使用 Cloudflare 的 NS 服务器。

    • Step1：登录 NameSilo 账号
    • Step2：打开域名管理
    • Step3: 更改 NS 服务器

Step1：登录 NameSilo 账号

打开官网，登录你的 NameSilo 账号（如果没有，就注册一个 NameSilo 账号）。

Step2：打开域名管理

打开域名管理页面，选中想要修改的域名。

Step3: 更改 NS 服务器

点击 “Change Nameserver（更改 NS 服务器）”，填写你想要设置的 NS 服务器地址，最后点击 "Submit（提交）" 按钮。
