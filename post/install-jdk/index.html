Java 入门指南（一）：JDK 的下载 & 安装

2019-03-08

本文是《Java 入门指南》的第一篇文章，介绍了 JDK 的下载 & 安装。

    • Step1：下载 JDK
    • Step2：安装 JDK
    • Step3：配置环境变量
    • Step4：测试环境变量是否配置成功

Step1：下载 JDK

Oracle 官网：https://www.oracle.com/java/technologies/downloads/

根据你的操作系统，下载对应平台的最新版本的 JDK。

比如说，我的操作系统是 Windows x64，下载的是 jdk-17_windows-x64_bin.exe。

Step2：安装 JDK

双击已下载的安装包，比如说 jdk-17_windows-x64_bin.exe，开始安装 JDK。

将 JDK 安装到指定的安装目录下，比如说 D:\App\Java\jdk-17.0.1。

Step3：配置环境变量

依次打开 此电脑→属性→高级系统设置→高级→环境变量(N)，开始配置环境变量。

在系统变量下，新建一个环境变量。变量名为 JAVA_HOME，变量值为 JDK 的安装目录，比如说

D:\App\Java\jdk-17.0.1\

在系统变量下，编辑 Path 环境变量。在顶部添加下面的一行，然后保存。

%JAVA_HOME%\bin

Step4：测试环境变量是否配置成功

打开命令提示符，输入下面的代码，然后回车。

java -version

如果出现了类似下面的提示，则说明环境变量已经配置成功。

java
